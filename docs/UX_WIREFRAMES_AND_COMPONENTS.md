## 5. Interaction Specifications

### 5.1 Hover States

| Element | Hover Behavior |
|---------|---------------|
| Value bars | Show exact amount + percentage in tooltip |
| Flow diagram nodes | Highlight connected paths |
| Metric cards | Subtle shadow lift (translateY -2px) |
| Leaderboard rows | Background highlight + "View Details" appears |

### 5.2 Click Actions

| Element | Click Action |
|---------|-------------|
| Value breakdown bar | Expand detailed breakdown panel |
| Flow diagram node | Filter to show only connected paths |
| Provider in leaderboard | Navigate to full valuation results |
| Metric card | Open drill-down modal with history |

### 5.3 Animation Timing

| Animation | Duration | Easing |
|-----------|----------|--------|
| Number counter | 800ms | ease-out |
| Bar growth | 600ms | ease-in-out |
| Card entry | 300ms | ease-out |
| Tooltip appear | 150ms | ease-out |
| Flow highlight | 200ms | ease-in-out |

- Collapse flow diagram to simplified list view
- Hide comparison explanations (show on tap)
- Full-width metric cards

### Tablet (640px - 1024px)
- 2-column grid for metric cards
- Side-by-side comparison (scrollable)
- Simplified flow diagram

### Desktop (> 1024px)
- Full layout as wireframed
- 4-column metric cards
- Full flow diagram with animations
- Side-by-side comparisons

## 7. Accessibility Requirements

### WCAG 2.1 AA Compliance

1. **Color Contrast**
   - All text meets 4.5:1 contrast ratio
   - Use patterns/icons in addition to color for status

2. **Keyboard Navigation**
   - All interactive elements focusable
   - Logical tab order
   - Visible focus indicators

3. **Screen Reader Support**
   - Proper ARIA labels on all components
   - Chart alternatives (data tables)
   - Live regions for dynamic updates

4. **Motion**
   - Respect prefers-reduced-motion
   - Pause/stop animated elements
